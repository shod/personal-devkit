# Code Review Report Template

> **This template is used by the `review-agent` subagent** to generate a report file.
> The subagent aggregates results from `/pint`, `/larastan`, and `/review` into a single document.

Use this template to generate the report. Replace `{placeholder}` with real values.

---

## Completion Rules

1. **Automated checks** — Pint and LaraStan results from skill output
2. **Summary table** — fill in the number of issues per category and severity
3. **Detailed report** — each issue as a separate block with file, line, description, and recommendation
4. **Severity in heading** — use `[CRITICAL]`, `[WARNING]`, `[INFO]`
5. **Summary** — all CRITICAL as "MUST FIX", WARNING as recommendations, plus positives
6. **Empty categories** — write "No issues found", do not remove the section
7. **Report filename**: `review-{task-id|diff}-{YYYY-MM-DD}.md`
8. **Path**: `specs/FS-*/reviews/`
