# laravel-kit

Laravel expertise, reusable across Laravel backends.

## Skills (populated)
- **laravel-best-practices** — 20 rule files (eloquent, queries, caching, security,
  testing, migrations, jobs, scheduling, validation, …) + SKILL.md
- **configuring-horizon** — full Horizon lifecycle (install, supervisors, metrics,
  notifications, tags). Uses `vendor/bin/sail artisan …` (both projects run Sail)
- **larastan** — PHPStan/LaraStan static analysis (horoscope targets level ≥ 8)
- **pint** — Laravel Pint formatting gate

matches this kit 1:1 (per horoscope constitution + backend-plan).
